# setup_env
Bunch of scripts and config files to setup a new environment 

## Install dependencies
Install dependencies on the system
### Ubuntu 
`sh -c "$(curl -sS https://raw.githubusercontent.com/nicolascochin/setup_env/master/install_deps_ubuntu.sh)" -- -h`

## Init env
Setup config files

`python3 -c "$(curl -sS https://raw.githubusercontent.com/nicolascochin/setup_env/master/init_env.py)" --help`
